The tutorial illustrations were created using OmniGraffle.

Each image was exported from .graffle to .png format.

Each .png image was processed in Adobe Photoshop Elements as follows:

    - Cropped
    - Image Size (reduced) to 660 pixels using Bicubic sharper (best for reduction)*
    - Saved For Web (images saved for web renamed with "_final".png.

*The maximum width for pages on the web site is currently 685 pixels.
